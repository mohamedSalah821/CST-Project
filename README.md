# Multi-Actor E-commerce System (Frontend)

## 📌 Project Overview
This project is a frontend-only e-commerce system built using:
- HTML
- CSS
- JavaScript
- Bootstrap

The system supports three types of users:
- Customers
- Sellers
- Admins

Each role has its own pages and UI functionalities.

---

## 🧱 Project Structure
The project is organized as follows:

- `index.html` : Home page
- `login.html` / `register.html` : Authentication pages

### Customer Pages
- `products.html` : Browse all products
- `product-details.html` : View product details
- `cart.html` : Shopping cart
- `checkout.html` : Checkout process

### Seller Pages
- `dashboard.html` : Seller dashboard
- `products.html` : Manage products
- `orders.html` : Manage orders

### Admin Pages
- `dashboard.html` : Admin dashboard
- `users.html` : Manage users
- `products.html` : Moderate products
